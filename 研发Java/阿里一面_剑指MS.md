（2）JVM如何加载一个类的过程，双亲委派模型中有哪些方法？
* 类加载过程主要分成三个步骤：加载、链接、初始化
* 加载：将字节码数据从不同的数据源（class文件、jar文件或者网络）读取到jvm，并映射为JVM认可的数据结构（Class对象）
* 链接：原始的类定义信息转换到JVM,具体分成三个步骤
    - 验证：JVM检验字节信息是否符合Java虚拟机规范，不符合就报错。
    - 准备：创建类或接口的静态变量，并初始化静态变量
    - 解析：将常量池的符号引用替换成直接引用，直接引用是通过对符号的解析，找到引用的实际地址
* 初始化：真正执行类初始化的逻辑，包括静态字段赋值，以及执行静态代码块。
